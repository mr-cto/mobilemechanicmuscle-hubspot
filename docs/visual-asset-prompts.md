## Production Notes

- **For Midjourney:** Append `--ar 16:9 --style raw --s 250` for cinematic realism
- **For DALL-E 3:** Use the prompts as-is, specify "photorealistic" if results skew illustrative
- **For Runway Gen-3 / Kling:** Use the video prompts, set duration to 10s, select "cinematic" style
- **Color consistency:** All assets should share the same palette — deep blacks (#0a0a0a), warm gold highlights (#c9a962), cool neutral midtones. Desaturated, editorial feel.
- **No brand logos:** Avoid generating identifiable brand logos/badges to prevent trademark issues. The silhouette shapes are recognizable enough.
- **File naming convention:**
  - `hero-bg.jpg` / `hero-bg.mp4`
  - `imgbreak-enginebay.jpg`
  - `stats-texture.jpg`
  - `make-porsche.jpg`, `make-bmw.jpg`, etc.
  - `svc-brakes.jpg`, `svc-diagnostics.jpg`, etc.
  - `cta-bg.jpg`
  - `exhaust-idle.mp3`

## 8. EXHAUST AUDIO — Scroll-Reactive Engine Rev

**Where it goes:** Loaded by the Web Audio engine and plays in the background when the user enables the sound toggle. Scroll speed modulates the playback rate (pitch) to simulate RPM changes.

**What it is:** A short (3–8 second) seamless loop of a high-performance engine at a steady mid-range RPM. The Web Audio API handles the idle-to-redline effect by slowing down or speeding up the playback rate, so the source file should be recorded at a consistent mid-range RPM for the cleanest pitch-shifting.

**Best source options (ranked):**

1. **Freesound.org** — Search for "V10 exhaust loop", "Lamborghini idle", "Ferrari rev loop", "supercar exhaust". Filter by Creative Commons license. Download as WAV, trim to a clean loop, export as MP3 at 192kbps.

2. **AI audio generation (ElevenLabs SFX, Stable Audio, Udio SFX):**
   > `Steady mid-RPM exhaust note from a naturally aspirated V10 supercar. Rich harmonic tone with slight mechanical rattle and deep bass resonance. Clean consistent RPM, no acceleration or deceleration. Seamless loop, 5 seconds. High fidelity engine recording, no background noise.`

3. **Sample from royalty-free SFX libraries** — Epidemic Sound, Artlist, Soundsnap. Search "supercar engine loop" or "exotic car exhaust steady state."

**Audio requirements:**
- **Format:** MP3 (192kbps+) or WAV
- **Duration:** 3–8 seconds (will be looped seamlessly by Web Audio)
- **Content:** Steady-state RPM, NOT an accel/decel sweep — the playback rate modulation handles that
- **Clean loop:** Trim so the waveform at the start and end match to avoid clicks. Crossfade 30-50ms at the loop point.
- **No reverb/room sound:** Dry, close-mic'd exhaust. The browser's audio graph handles any processing.
- **Filename:** `exhaust-idle.mp3`
- **File size target:** Under 500KB for fast load
